# AgriPredict Frontend

A premium React + Vite frontend for the AgriPredict crop yield prediction platform.

## Overview

This UI includes:

- modern dashboard layout with glassmorphism styling
- responsive predictor form for crop, soil, climate, and nutrient inputs
- prediction results panel with yield, confidence, and recommendations
- analytics cards and premium SaaS-inspired visuals
- mobile-first design using `Poppins` font

## Scripts

- `npm install` — install frontend dependencies
- `npm run dev` — start the Vite development server
- `npm run build` — build the production frontend bundle
- `npm run preview` — locally preview the production build

## Notes

The frontend posts prediction requests to the backend API at `VITE_API_URL` or `http://localhost:5000` by default.
